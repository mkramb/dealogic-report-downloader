# Dealogic report downloader

## Prerequisite

Install python & [poetry](https://python-poetry.org/), needed for docs diagrams:

```
curl -sSL https://install.python-poetry.org | python3 -

poetry install
```

Now download browser binaries and their dependencies:

```
playwright install
```

## QuickStart

Execute downloader:

```
./scripts/start.sh
```
